# Что есть на сайте?

## Нотации


$1

Это нотация игры. Она состоит из раздвигающихся панелей. Чтобы посмотреть нотацию нужно по ней кликнуть, тогда откроются ходы нотации. По ходам тоже можно кликать. По клику на ход загрузится соответствующая ему доска. У ходов есть варианты как, например, у первого хода сделан альтернативный ход - $1:1/1@2 вместо ошибочного: $1:1/0@2 .

По ходам также можно кликать.
